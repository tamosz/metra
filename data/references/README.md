# Forum References

Extracted knowledge from royals.ms forum threads, organized by topic. These files
capture stable facts (formulas, balance values, mechanics) so they don't need to be
re-fetched every session.

## How to use

Check these files BEFORE web-searching royals.ms. Most game mechanics and balance
values are already documented here.

## How to add entries

Use this format:

```markdown
## Section Name

**Source:** [Thread Title](https://royals.ms/forum/threads/...)
**Accessed:** YYYY-MM-DD
**Used in:** `data/skills/foo.json`, `src/engine/bar.ts`

- Extracted fact 1
- Extracted fact 2
```

Keep entries factual — values, formulas, and brief quotes. Skip opinions and discussion.
